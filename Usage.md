# Usage #
```

tommy.pl <mode> [<options>...] [Note Title]

Modes:
--append      Add text to the end of an existing note
--create      Create a new note
--display     Show the contents of a note
--edit        Edit an existing note
--list        Show the titles of recent notes
--search      Search for words in notes

options:
--gui               Use the tomboy GUI for the requested action
--tag=<string>      Search for or add these tags to the note
--text=<string>     Use this text instead of interactive prompt
--xml               Show or edit note xml
--startdate=<date>  Only show notes after this date
--enddate=<date>    Only show notes before this date
--count=<int>       Return at most this number of notes (default 10)

```
